# RRIMS Frontend

Advanced React + Tailwind frontend for the RRIMS backend hosted at `https://pantasaugat.com.np`.

## Run

```bash
npm install
npm run dev
```

The API base URL is configured in `.env`:

```bash
VITE_API_BASE_URL=https://pantasaugat.com.np/api/v1
```

## Build

```bash
npm run build
```
